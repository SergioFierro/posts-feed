## About
App that lists all posts and their details from [JSONPlaceholder](https://jsonplaceholder.typicode.com/). Project built using MVVM architecture, with daggerHilt, retrofit, room, coroutines.

<a name='Requirements'></a>

## Requirements
* Android Studio 4.0 or higher
* Java 8
* Android 6.0 (23) SDK or higher
* Gradle 6.1.1
* Kotlin 1.4.10
